# 🌤️ Weather API App

A beginner-friendly Python project that fetches live weather data using the OpenWeatherMap API.

## Features

- Search weather by city
- Current temperature
- Feels like temperature
- Humidity
- Wind speed
- Atmospheric pressure
- Weather description
- Error handling

## Requirements

- Python 3
- requests

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python weather.py
```

## API

This project uses the OpenWeatherMap API.

Replace:

```python
API_KEY = "YOUR_API_KEY"
```

with your own API key.
